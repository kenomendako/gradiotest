## メモの書き方

```markdown
- [ ] やりたいこと / 問題の説明
  - 詳細があれば追記
  - スクリーンショットがあればパスを記載
```

- [ ] チャット入力欄のShift＋Enterで送信を廃止したい。Enter送信のみ。もしくはCtrl＋Enterで送信。コーディングエディタの操作（Shift＋Enterで改行）と間違えて改行したいところで送信してしまう事故頻発するため。
- [ ] ルームの個別設定保存ボタンを削除。（基本は即時保存のはずなので不要。ボタンでの保存が必要な個所はそれぞれに保存ボタンを設置）
